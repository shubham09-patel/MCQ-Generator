# MCQ Generator

An AI-powered PDF to MCQ Generator built using Python, Streamlit, and NLP techniques. The application extracts text from PDF files and automatically generates multiple-choice questions.

## Features
Upload PDF files
Extract text from PDFs
Generate MCQs automatically
Answer evaluation system
Score calculation
Download MCQs as text file

## Technologies Used
Python
Streamlit
SpaCy
PyMuPDF
NLP

## Run Project

streamlit run dcs.py
